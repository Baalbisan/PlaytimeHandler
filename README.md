# Playtime Handler
A simple program written in C to display steam game playtime in fastfetch.  
  
Please check the wiki page for usage instructions.

# Installation
This application depends on GCC, so make sure it is installed.
Clone the repo and run make.  
You could optionally add the alias to your `~/.bashrc`:
```
alias fastfetch='./path/to/ff_pt_handler && fastfetch'
```

# libraries used:
- [DaveGamble/cJSON](https://github.com/DaveGamble/cJSON)
- [DreamyCecil/VDF](https://github.com/DreamyCecil/VDF)
